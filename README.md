# RangeVariable
Easily clamp the values in min and max inclusively using Range Variable.

Currently has:
- RangeInt
- RangeFloat
- RangeDouble
- RangeLong
- RangeByte

The .dll file can be used inside Unity.

Or you can dowwnload this repository, and drag-n-drop these files to your projects.
